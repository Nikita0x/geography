# Project Instructions

Это проект на **Flutter**, через который пользователь хочет:

-   Учить и практиковать Flutter на реальном приложении.
-   Постепенно довести приложение до продакшена.
-   Выложить готовое приложение в Google Play.

Идея приложения:

-   Это учебное приложение по **изучению географии**.
-   В нём планируется несколько режимов, например:
    -   режим флагов (отображается название страны — нужно угадать флаг);
    -   обратный режим (показывается флаг — нужно угадать страну);
    -   другие игровые режимы в похожем духе.
-   В перспективе планируется подключить **рекламу** и попробовать **монетизацию**.
-   Это **личный, первый полностью соло‑проект** пользователя, поверх уже имеющегося опыта разработки мобильных приложений (~2 года).

Ассистенту (Cascade) важно:

-   Объяснять концепции Flutter простым языком, с привязкой к текущему коду.
-   Помогать пошагово развивать проект, не ломая существующую структуру.
-   Учитывать, что цель — не только результат, но и обучение пользователя.

Технические предпочтения пользователя:

-   Для управления состоянием по возможности использовать **signals** во Flutter.
-   Даже если этот паттерн не самый популярный в Flutter-экосистеме, он привычен пользователю из мира фронтенда/JS, поэтому при наличии выбора предпочтителен именно он.
